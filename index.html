1. Menerima Perintah Web (RX)

Susun blok MakeCode persis seperti logika di bawah ini. Ingat, gunakan blok parse to number untuk mengubah teks menjadi angka, dan blok else if agar komponen tidak saling mengganggu!

bluetooth.onUartDataReceived(serial.delimiters(Delimiters.NewLine), function () {
    // 1. Baca NAMA komponen sampai ketemu KOMA (,)
    nama = bluetooth.uartReadUntil(serial.delimiters(Delimiters.Comma))
    
    // 2. Baca ANGKA sampai ketemu ENTER (NewLine) lalu ubah jadi angka beneran
    angka = parseFloat(bluetooth.uartReadUntil(serial.delimiters(Delimiters.NewLine)))
    
    // 3. Pisahkan kamarnya! Lampu masuk ke kamar lampu, Kipas masuk ke kamar kipas
    if (nama == "lampu") {
        // Karena switch bernilai 0 atau 1, langsung masuk pin Digital
        pins.digitalWritePin(DigitalPin.P1, angka)
    } else if (nama == "kipas") {
        // Karena dimmer bernilai 0-1024, langsung masuk pin Analog
        pins.analogWritePin(AnalogPin.P2, angka)
    }
})

2. Mengirim Sensor ke Web (TX)

Web akan otomatis membaca data ini dan membuatkan panel grafik sensor secara otomatis!

basic.forever(function () {
    // Akan otomatis menjadi key: "Sensor cahaya" di Firebase dan Web
    bluetooth.uartWriteValue("Sensor cahaya", pins.analogReadPin(AnalogPin.P0))
    basic.pause(500)
})
